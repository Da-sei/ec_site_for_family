# プロダクト概要

家族間の物品譲渡・フリマプラットフォーム。家族メンバーがグループを作成・参加し、不用品を出品してリクエストベースで譲渡するモバイルアプリ＋APIシステム。

## コア機能

- **認証**: アカウント登録・JWTログイン
- **グループ管理**: グループ作成・招待トークンによるメンバー参加
- **商品管理**: グループ内への商品出品・編集・削除・画像添付
- **リクエスト処理**: 商品への譲渡リクエスト・承認/拒否/完了
- **ウィッシュリスト**: グループ内で欲しいものを登録・共有（タイトル・説明・作成者・グループ紐付け）

## 主なユースケース

- ユーザーがグループを作成し、招待リンクで家族を招待する
- グループメンバーが不用品を出品（カテゴリ・配送方法・画像付き）
- 欲しい商品にリクエストを送り、出品者が承認/拒否する
- リクエスト完了で商品ステータスが `TRANSFERRED` に変わる

## バリュープロポジション

家族専用のクローズドなフリマとして、グループ内の信頼関係に基づいたシンプルな物品譲渡を実現。NestJS REST API + Flutter モバイルアプリの構成により、iOS/Android 両対応。
